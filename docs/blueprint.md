# **App Name**: PoshanAI

## Core Features:

- User Authentication: Secure user signup, login, and logout functionality with JWT-based authentication.
- Health Onboarding Wizard: Multi-step onboarding process to collect user's age, gender, blood group, allergies, weight, and height.
- AI Health Report Analysis: Analyze uploaded health reports (PDF/images) using OCR to extract key health markers.
- Personalized Plan Generation: AI tool that generates personalized daily diet and fitness plans based on user profile and health data.
- Meal Image Analysis: AI tool to identify food items from uploaded meal images, estimate calories & nutrients, identify allergens, and suggest improvements.
- Daily Progress Tracking: UI for users to track meals, exercise, and hydration, with a dashboard displaying total calories and nutrient scores.
- Admin Panel: Admin interface to manage food items, recommendation rules, and user health summaries.

## Style Guidelines:

- Primary color: Vibrant cyan (#48cae4) to evoke a sense of health and vitality.
- Background color: Light cyan (#e0f7fa), a desaturated variant of the primary color to create a calm backdrop.
- Accent color: Soft blue (#90e0ef), analogous to cyan, providing visual interest while maintaining harmony.
- Body and headline font: 'Inter', a sans-serif font for clear and modern readability.
- Use clear, friendly icons representing different food groups, exercises, and health metrics.
- Clean and intuitive layout with a focus on easy navigation and data visualization.
- Subtle transitions and animations to provide feedback and enhance user experience.
